Objeto Math

O objeto Math fornece propriedades e métodos matemáticos. Os valores são disponíveis para métodos e constantes.

Lista de propriedades do objeto Math:

LN10- Logaritmo natural de 10;

LN2- Logaritmo natural de 2;

LOG10E- Logaritmo de base 10 de E;

PI- Valor de PI;

SQRT2- Raiz quadrada de 2;

SQRT1_2- Raiz quadrada de 1/2;

Ceil(x)- Retorna o valor de x arredondado para cima;

Floor(x)- Retorna o valor de x arredondado para baixo;

Pow - Eleva um número a qualquer potência;

Random- Retorna um valor aleatório;

Objeto Date

Objeto Date é um objeto embutido do JavaScript que permite trabalhar com datas e horas.

Os objetos Date somente podem ser instanciados em Js usando o comando new.

Sintaxe criação objeto Date:

var data = new Date()

var data = new Date(milissegundos)

var data = new Date(ano, mês, dia)

var nome = new Date(string_de_data)
