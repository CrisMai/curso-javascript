Arrays/Vetores

Variáveis compostas armazenam vários valores em uma mesma estrutura.

Os arrays em JS são utilizados para armazenar vários valores e têm diversos compartimentos chamados de posições.

Cada item de dado armazenado em uma posição do array é chamado de elemento.

Algumas propriedades:

.push(): permite acrescentar valores em um array.

.length: permite saber o comprimento de um array. Não tem parêntese após a palavra length.

.sort(): coloca em ordem crescente os elementos de um array.

.indexOf(): comando que permite buscar valores dentro de um array.

.pop(): remove o último elemento de um array.

.shift(): remove o primeiro elemento de um array.

.splice(): permite inserir um elemento em uma posição específica.

.concat(): permite concatenar dois ou mais arrays e formar um novo.

.reverse(): ordena os elementos em ordem descendente de posições.
