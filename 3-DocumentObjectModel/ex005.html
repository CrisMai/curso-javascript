Iniciando estudos com DOM

DOM - Document Object Model é um conjunto de objetos dentro do navegador que dará acesso aos componentes internos do website.

Árvore DOM começa da raiz e é chamada de window. Elementos são quaisquer estruturas que aparecem dentro da árvore DOM e é possível selecionar esses elementos para poder navegar dentro da árvore DOM

Existem vários métodos para selecionar: por marca, por ID, por nome, por classe, por seletor.
