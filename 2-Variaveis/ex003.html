Conversão de Tipos

Em alguns casos é necessário fazer a conversão de tipos, que pode ser de uma string para um número.

Number.parseInt(n) ==> faz a conversão de um número para um número inteiro.

Number.parseFloat(n) ==> conversão de um número para um número real.

Nas versões mais atuais pode ser usado somente Number(n) onde o JavaScript vai fazer a conversão de acordo com os dados.

Conversão de um número para string: string(n) ou n.tostring().

Para formatação de strings usa `TEMPLATE STRING´ que é o símbolo ==> ${...}.
